# Joeydash_bot

#### A web implementation of chatterbot trained using whatsapp group messages and hosted using Flask.

## Local Setup:
 1. run `pip install -r requirements.txt`
 2. Run *app.py*
 3. Demo will be live at [http://localhost:3000/](http://localhost:3000/)
